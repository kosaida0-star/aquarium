Explanation of Changes

In my version of the program I modified several parameters of the mathematical system to change the shape, movement, and appearance of the organism. The program uses trigonometric functions such as sin(), cos(), and atan2() to transform normal (x, y) coordinates into a curved organic structure. These functions create wave patterns that generate the shape of the organism.

First, I increased the canvas size to 600 × 450 so the drawing has more horizontal space. I also used background(255, 8) instead of a fully solid background. The second value adds transparency, which means the previous frame is not completely erased. Because of this, the moving points leave a fading trail behind them, creating a tail-like effect.

I increased the probability of drawing points by changing the random condition to random() < 0.007. This produces more points on each frame, making the organism appear denser and smoother.

To modify the shape, I stretched the horizontal coordinate by multiplying the x component by 3 in the equation xCoord = (q + wave) * 3 * sin(c). This makes the figure wider and creates a horizontal oval shape. I also changed the rotation formula to c = d / 7 + e / 333 - t / 70, which slows the rotation and produces a smoother movement.

Finally, I increased the animation speed by changing the time variable to t += 3.3. The variable t controls how the sine waves evolve over time, so increasing it makes the organism move faster while maintaining its flowing motion.
